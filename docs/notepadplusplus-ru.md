## Введение

**Notepad++** — это бесплатный текстовый редактор и редактор исходного кода с открытым исходным кодом, работающий в среде Microsoft Windows. Он распространяется под лицензией GNU General Public License (GPL) и является одним из наиболее популярных редакторов среди разработчиков, системных администраторов и обычных пользователей.

Название проекта намекает на то, что это «улучшенная версия» стандартного Блокнота (Notepad), встроенного в Windows — и это действительно так: Notepad++ предлагает значительно более богатый функционал при такой же лёгкости и скорости работы.

---

## История

Проект был создан французским разработчиком **Don Ho** и впервые выпущен в **2003 году**. С тех пор редактор активно развивается силами сообщества и регулярно обновляется. По данным опросов разработчиков Stack Overflow, Notepad++ на протяжении многих лет удерживал звание самого популярного текстового редактора в мире.

## Ключевые возможности

### Подсветка синтаксиса
Notepad++ поддерживает подсветку синтаксиса более чем для **80 языков программирования и разметки**, включая C, C++, C#, Java, JavaScript, Python, HTML, XML, SQL, Markdown и многие другие. Пользователи также могут создавать собственные правила подсветки через **User Defined Language (UDL)**.

### Редактирование нескольких документов
Редактор поддерживает работу с несколькими файлами одновременно через систему **вкладок (буферов)**. Окно можно разделить на два вида (Main View и Sub View), открыв один и тот же или разные файлы бок о бок.

### Умные возможности редактирования
- **Автодополнение** слов и функций
- **Сворачивание кода** (code folding) по блокам
- **Поиск и замена** с поддержкой регулярных выражений
- **Мультикурсорное редактирование** и колоночный режим выделения
- **Макросы** — запись и воспроизведение последовательностей действий
- **Сравнение файлов** (через плагин Compare)

### Работа с кодировками и форматами
Notepad++ полностью поддерживает **Unicode (UTF-8, UTF-16)**, а также множество других кодировок. Поддерживается конвертация форматов окончания строк: Windows (CRLF), Unix (LF) и macOS (CR).

### Плагины
Одна из главных сил Notepad++ — развитая **экосистема плагинов**. Через встроенный **Plugin Manager** можно установить сотни расширений: от интеграции с системами контроля версий до инструментов для работы с базами данных, XML, JSON и многим другим.

## Архитектура

Нotepad++ построен на базе компонента **Scintilla** — мощного редактирующего движка с открытым кодом, который обеспечивает всю низкоуровневую работу с текстом: подсветку синтаксиса, сворачивание кода, маркеры и многое другое. Взаимодействие с плагинами происходит через Plugin API на основе Windows-сообщений (`SendMessage`), что делает разработку плагинов доступной на любом языке, поддерживающем Win32 API.

## Для кого подходит Notepad++?

| Аудитория | Применение |
|---|---|
| Разработчики | Редактирование кода, работа с конфигами, скриптами |
| Системные администраторы | Анализ логов, редактирование конфигурационных файлов |
| Веб-мастера | Редактирование HTML, CSS, JS, PHP |
| Обычные пользователи | Замена стандартного Блокнота с расширенными возможностями |

## Лицензия и открытый код

Notepad++ полностью **бесплатен** и распространяется под лицензией **GPL v3**. Исходный код открыт и доступен на GitHub. Любой желающий может изучить код, внести исправления или создать форк проекта.
